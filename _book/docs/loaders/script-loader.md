## 安装

```bash
npm install --save-dev script-loader
```

## 用法

在全局上下文(global context)执行一次 JS 脚本。

> :警告: 在 node.js 中不会运行

### 配置（推荐）

```js
import exec from 'script.exec.js';
```

**webpack.config.js**
```js
module.exports = {
  module: {
    rules: [
      {
        test: /\.exec\.js$/,
        use: [ 'script-loader' ]
      }
    ]
  }
}
```

### 内联

```js
import exec from 'script-loader!./script.js';
```
