## Examples

**webpack.config.js**

```js
{
  test: /.../,
  use: ExtractTextPlugin.extract({
    use: [ ... ],
    fallback: [
      'dynamic-css-loader',
      'file-loader',
      'extract-loader',
    ],
  }),
},
```

```js

```

