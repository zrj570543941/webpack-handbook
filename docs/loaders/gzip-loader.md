## Install

```bash
npm install --save-dev gzip-loader
```

## Usage

**webpack.config.js**

```js
module.exports = {
  module: {
    rules: [
      {
        test: /\.gz$/,
        enforce: 'pre',
        use: 'gzip-loader'
      }
    ]
  }
}
```

**bundle.js**

```js
require("gzip-loader!./file.js.gz");
```
